# 7. CSS Concepts Used

## External Stylesheet

A separate style.css file is linked with the HTML document to manage all website styling.

## CSS Selectors

The following selectors are used:

### Element Selector

header {
background-color: #2c3e50;
}

### Class Selector

.skill-card {
padding: 15px;
}

### ID Selector

#about {
padding: 30px;
}

## Box Model

Used:

* Margin
* Padding
* Border

to create spacing and structure.

## Flexbox

Used for:

* Navigation menu
* Skills section layout

Benefits:

* Easy alignment
* Responsive layouts
* Flexible spacing

## Hover Effects

Implemented on:

* Navigation links
* Submit button

Purpose:

* Improve user interaction
* Provide visual feedback

## Media Queries

Used to create a mobile-friendly design.

Example:

@media (max-width: 768px)

This adjusts the layout for smaller screen sizes.
